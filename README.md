## üß† Strategy Logic (50 / 200 SMA Crossover)

The strategy is a simple, long-only, trend-following model designed to capture large directional moves while minimizing exposure during extended consolidation or bear markets.

* **Target Asset:** Apple Inc. (Ticker: `AAPL`)
* **Timeframe:** Daily candles (`D`)
* **Entry Signal (Go Long):** The **Fast SMA (50 periods)** crosses **ABOVE** the **Slow SMA (200 periods)**.
* **Exit Signal (Go Flat):** The **Fast SMA (50 periods)** crosses **BELOW** the **Slow SMA (200 periods)**.
